    Line Plots¶

    Los gráficos de línea (line plot) son una de las formas más comunes de visualizar datos. Quizás la más simple de todas las gráficas es la visualización de una sola función $y = f(x)$.

    In [1]:
    Copied! 
    import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5]
y = [1, 4, 2, 5, 3]

plt.plot(x, y)
plt.show()
    import matplotlib.pyplot as plt x = [1, 2, 3, 4, 5] y = [1, 4, 2, 5, 3] plt.plot(x, y) plt.show()

    En este ejemplo, se están trazando dos listas de valores, x e y, donde los valores de x representan los puntos en el eje x y los valores de y representan los puntos en el eje y. La función plot se utiliza para trazar los puntos en un gráfico de línea.

    Puede personalizar el gráfico de línea de varias formas, incluyendo la elección del color y estilo de línea. Por ejemplo, puede usar la cadena de formato 'ro-' para dibujar puntos rojos interconectados con una línea sólida:

    In [2]:
    Copied! 
    import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5]
y = [1, 4, 2, 5, 3]

plt.plot(x, y, 'ro-')
plt.show()
    import matplotlib.pyplot as plt x = [1, 2, 3, 4, 5] y = [1, 4, 2, 5, 3] plt.plot(x, y, 'ro-') plt.show()

    También puede agregar etiquetas de eje y un título al gráfico utilizando las funciones xlabel, ylabel y title:

    In [3]:
    Copied! 
    import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5]
y = [1, 4, 2, 5, 3]

plt.plot(x, y, 'ro-')
plt.xlabel('Eje X')
plt.ylabel('Eje Y')
plt.title('Gráfico de línea')
plt.show()
    import matplotlib.pyplot as plt x = [1, 2, 3, 4, 5] y = [1, 4, 2, 5, 3] plt.plot(x, y, 'ro-') plt.xlabel('Eje X') plt.ylabel('Eje Y') plt.title('Gráfico de línea') plt.show()

    Matplotlib también le permite trazar varias líneas en un solo gráfico utilizando la función plot varias veces. Puede usar la función legend para agregar una leyenda que indique qué línea representa cada conjunto de datos:

    In [4]:
    Copied! 
    import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5]
y1 = [1, 4, 2, 5, 3]
y2 = [3, 1, 4, 2, 6]

plt.plot(x, y1, 'ro-', label='Conjunto de datos 1')
plt.plot(x, y2, 'bs-', label='Conjunto de datos 2')
plt.xlabel('Eje X')
plt.ylabel('Eje Y')
plt.title('Gráfico de línea')
plt.legend()
plt.show()
    import matplotlib.pyplot as plt x = [1, 2, 3, 4, 5] y1 = [1, 4, 2, 5, 3] y2 = [3, 1, 4, 2, 6] plt.plot(x, y1, 'ro-', label='Conjunto de datos 1') plt.plot(x, y2, 'bs-', label='Conjunto de datos 2') plt.xlabel('Eje X') plt.ylabel('Eje Y') plt.title('Gráfico de línea') plt.legend() plt.show()

    En este ejemplo, se están trazando dos conjuntos de datos, y1 y y2, y se ha agregado una leyenda que indica qué conjunto de datos representa cada línea.
